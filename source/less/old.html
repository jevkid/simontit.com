Projects

ONTarget

For my 2nd year group project, we were required to create a web-based service using image or audio analysis. My team and I chose image analysis, and we created a social media website for archery clubs across the UK. Users were able to upload photos of their targets, and our service would automatically analyse the score based on where the arrow was, and additional details the user had entered. Teams could also keep track of different players' scores using a leaderboard system. The code can be found on my GitHub, and screenshots are under Web Design below.

IBM Smarter Planet Project

For my final year project, my team and I were given the opportunity to work with the Emerging Technologies group at IBM. This began in September 2015 and was completed by May 2016. IBM gave us one goal: to develop and implement a system that promoted a "smarter planet" and involved data analytics. Using this, we came up with the idea to develop a sensor-based system that could collect and analyse data from student accommodation in order to provide real-time information, alerts, and data analytics pertaining to utility usage and security. Using a Raspberry Pi, several affordable sensors, and Sen.se's Mother, we were able to accomplish these goals. To read more about the project, you can check out my dissertation here. If you'd like to see the code that was developed, it can be viewed on my GitHub Repository.
